Music Tools

free browser-based tools for musicians

A small collection of free, open-source, browser-based tools for musicians at any level. No account, no installation, no server — each tool is a single HTML file that runs entirely in your browser, powered by the Web Audio API.

Vanilla HTML · CSS · JS Web Audio API Zero Dependencies CC BY-NC 4.0
Ear Trainer

Sharpen your musical ear with interactive scale, interval, and chord quizzes across 15 scales. Visualise notes on piano keyboard, treble staff, guitar, bass, and ukulele fretboards.

A precise chromatic strobe tuner for guitar, bass, ukulele, and any instrument. Professional-grade accuracy with an animated strobe display, needle cent meter, and 17 instrument presets.

A visual metronome with tap tempo, subdivisions, swing, and an animated pendulum arm. Keep perfect time with per-beat accent control, 8 time signatures, and a click shaper.

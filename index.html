Music Tools

free browser-based tools for musicians

A small collection of free, open-source, browser-based tools for musicians at any level. No account, no installation, no server — each tool is a single HTML file that runs entirely in your browser, powered by the Web Audio API.

Vanilla HTML · CSS · JS Web Audio API Zero Dependencies CC BY-NC 4.0

Instrument Tuner

A precise chromatic tuner for guitar, bass, ukulele, and any instrument. Features a scrolling pitch history trace, EMA-smoothed needle cent meter, input level bar, reference tone playback, and 16 tuning presets.

Pulse

A visual metronome with tap tempo, subdivisions, swing, and an animated pendulum arm. Keep perfect time with per-beat and per-subdivision accent control, 8 time signatures, and a click shaper.
